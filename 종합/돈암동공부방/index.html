돈암동 공부방
이 모든 활동은 단기적인 성적 향상뿐 아니라 장기적으로 사고의 유연성과 문제 해결 능력을 키우는 데 기여하며 학교 교육과 일상이 자연스럽게 연결되는 학습의 새로운 질서를 만들어간다. 공부한 내용을 매일 정리하며 하루의 학습 기록을 단순한 체크리스트로 만들기보다는, 스스로에게 묻는 질문들을 담은 반성 기록으로 탈바꿈시킨다. 돈암동 공부방은 학생의 학습 동기 상태를 매주 점검하여, 외적 동기 중심인지 내적 동기 중심인지 유형을 분류하고, 각각에 맞는 동기부여 전략을 적용한다. 스스로 오답을 정리하고, 그것을 정기적으로 복습하는 습관은, 실수를 두려워하기보다는 학습의 소중한 자료로 여기게 만든다. 집중력이 저하될 경우 필기를 다시 정돈하도록 유도해 주의력을 회복하고, 유사 문제를 다양한 접근법으로 풀어보며 해법을 비교한다. 학습 시간을 계획할 때 단순히 ‘이 교과서 쪽을 몇 쪽까지 하기’처럼 목표를 정하는 대신, 이론 설명 후 즉각 실습 문제를 풀고, 실제 시험에서 접할 수 있는 변형된 유형까지 연결하는 구조로 회차를 설계하면 학습의 밀도가 달라진다. 돈암동 공부방은 긴 해설보다는 학생의 집중을 끌어당기는 짧고 강렬한 문구를 활용해 뇌의 인지 부담을 줄이고, 학습 흐름을 끊지 않도록 유도한다.
